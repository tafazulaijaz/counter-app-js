# Counter App

A simple counter app built with HTML, CSS, and Vanilla JavaScript.

## Features
- Increment / Decrement counter
- Reset functionality
- Clean UI
- Uses DOM manipulation & event listeners

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
Built to practice core JavaScript concepts like DOM selection and event handling.
